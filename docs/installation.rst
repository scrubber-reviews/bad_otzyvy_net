============
Installation
============


Stable release
--------------

To install  bad_otzyvy_net, run this command in your terminal:

.. code-block:: console

    $ pip install bad_otzyvy_net

This is the preferred method to install  bad_otzyvy_net, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for  bad_otzyvy_net can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/NMelis/bad_otzyvy_net

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/NMelis/bad_otzyvy_net/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/NMelis/bad_otzyvy_net
.. _tarball: https://github.com/NMelis/bad_otzyvy_net/tarball/master
